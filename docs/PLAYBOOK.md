# Implementation Playbook: NOVA Agent Logic

This document outlines the repeatable build process for the Mātauranga NOVA agent, designed for deployment within Claude-based environments.

## Phase 1: Identity & Tone Architecture
* **Objective:** Establish a non-clinical, peer-to-peer persona.
* **Logic:** Mirroring language (Spanish Rioplatense / NZ Casual) to build immediate trust.
* **Safety:** Integration of "Te Whare Tapa Whā" model for culturally safe interactions in Aotearoa.

## Phase 2: Agent Skills & Protocols
* **Crisis Management:** Implementation of a "Human-First Check-in" before resource delivery.
* **Moment Logic:** Six defined "Real Moments" to guide the conversation flow without rigid branching.

## Phase 3: Technical Integration
* **Model:** Claude 3.5 Sonnet (Optimized for nuance).
* **System Parameter:** High-priority instruction set (System Prompt V5).
* **Memory Hooks:** Designed for session-less continuity.
